Utilisation de la fenetre de training:

il vous faut dans un premier temps choisit sous fichier "continuer projet" ou "nouveau projet"
si vous selectionnez en tant que nouveau fichiers un projet que vous avez déjà selectionner vous serai renvoyer sur la list edes projet déjà commence 

le projet est sauvegardé lorsque vous apuiyez sur les touches "suivant et enregistrer", "precedents" ou "enregistrer" (dans la barre de menu) 

sur chacune des pages, il faut encadrer chacun des elements

par defaut les element crée sont des paragraphes, pour changer le type de l'element:
-selectionner l(es)'element(s) en question dans la list des actions
-selectionneravec le radiobouton le type voulu
-clicker sur "confirmer"

Pour crée le chemin de fer il faut que 20% des pages soit traité, si ce n'est pas fait, le traitement ne sera pas efficase, si les premieres pages ne sont pas representative (couverture,page vierge,...) ne les comptez pas dans les 20%

vous pouvez ensuite crée le chemin de fer: barre de menu>>chemin de fer>>crée

ATTENTION: une fois le chemin de faire crée il ne sert a rien de continue a modifer/ajouter des elements via la fenetre de training, cela n'aura auccun impact sur le chemin de fer

Si vous avez déjà crée le chemin de fer:
apuyez sur "chemin de fer>>cree" cela ne refera pas tout le traitementmais vous affichera simplement le resultat
