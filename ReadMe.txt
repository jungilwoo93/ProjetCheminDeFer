Utilisation de la fenêtre de training:

il vous faut dans un premier temps choisir sous fichier "continuer projet" ou "nouveau projet"
si vous selectionnez en tant que nouveau fichier un projet que vous avez déjà selectionné, vous serez renvoyés sur la liste des projets déjà commencés 

le projet est sauvegardé lorsque vous appuierez sur les touches "suivant et enregistrer", "précedents" ou "enregistrer" (dans la barre de menu) 

sur chacune des pages, il faut encadrer chacun des éléments

par défaut les éléments créés sont des paragraphes, pour changer le type de l'élément:
-sélectionner l(es)'élément(s) en question dans la liste des actions
-selectionner avec le radiobouton le type voulu
-cliquer sur "confirmer"

Pour créer le chemin de fer, il faut que 20% des pages soient traitées, si ce n'est pas fait, le traitement ne sera pas efficace, si les premières pages ne sont pas représentatives (couverture,page vierge,...) ne les comptez pas dans les 20%

vous pouvez ensuite créer le chemin de fer: barre de menu>>chemin de fer>>créer

ATTENTION: une fois le chemin de fer créé, il ne sert à rien de continuer à modifer/ajouter des éléments via la fenêtre de training, cela n'aura aucun impact sur le chemin de fer

Si vous avez déjà créé le chemin de fer:
appuyez sur "chemin de fer>>créé" cela ne refera pas tout le traitement, mais vous affichera simplement le résultat
