Utilisation du chemin de fer:

Modification de l'affichage:
-affichage>>rectangle
vous permet d'avoir des rectangles plein ou vide pour pouvoir faire des etude plus ou moins global 
-affichage>>dimention
permet de gére le nombre de page par ligne
c'est cette même dimention qui est utilisé pour la création du pdf


Export en pdf:
fichier>>exporter en pdf
le nombre d'image page par feuille est celui choisit dans dimention

Modification:
la modification permet de corriger un erreur du du au traitement 
pour ce faire clicker sur le bouton en haut à droite de la page à modifier puis clickez sur modification>>corriger une erreur

Recuperation des references de page:
clickez sur le bouton en haut à gauche de chaque page (même que bouton "modifier") vous povez ensuite effectuer un colle dans le document de votre choix

Deplacement dans le chemin de fer:
pour le deplacement vertical et horizontal utiliser les scrollbar
pour zoomer/dezoomer vous pouver le faire avec le pad ou avec la souris
